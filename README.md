## 下载

### Docker

```
docker pull apachecn0/javatpoint-php-zh
docker run -tid -p <port>:80 apachecn0/javatpoint-php-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g javatpoint-php-zh
javatpoint-php-zh <port>
# 访问 http://localhost:{port} 查看文档
```
