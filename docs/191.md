# Heredoc 语法

> 原文:[https://www.javatpoint.com/php-heredoc-syntax](https://www.javatpoint.com/php-heredoc-syntax)

通过使用语法，我们可以通过 PHP 脚本显示 HTML 元素。

## 句法

```
<<<name of string
	//content
name of string

```

## 例 1

```
<?php
	echo <<<javatpoint
	welcome to the best tutorial site
javatpoint;
?>

```
