# 布尔代数学体系的

> 原文:[https://www.javatpoint.com/php-boolean](https://www.javatpoint.com/php-boolean)

这些数据类型提供 0 或 1 输出。在 PHP 中，True 的值是 1，False 的值什么都不是。

## 句法

```
<?php
$foo = True; // assign the value TRUE to $foo
?> 

```

## 例 1

```
<?php
	$a=true;
	echo $a;
?>

```
