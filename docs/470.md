# 法尔孔特征

> 原文:[https://www.javatpoint.com/features-of-phalcon](https://www.javatpoint.com/features-of-phalcon)

*   它是 PHP 最快的框架。
*   这是一个全栈开源框架。
*   它可以创建独立的框架。
*   它需要更少的代码来实现特定的任务。
*   它提供对象关系映射、模板引擎、路由、缓存等。
*   它是一个**松散耦合的**框架，用户可以根据应用需求构建一个目录结构的项目。
